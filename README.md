# Plate-Harmony

## Getting started (Godot)

1. Install Godot 4.x (recommended 4.2 stable) from https://godotengine.org.
2. Open the project: `File -> Open` and select this repository's folder.
3. The main scene is `res://scenes/main.tscn`. Run the project to verify the scaffold.
4. Use Git LFS for large assets. See `.gitattributes` for recommended patterns.

If you're new to Godot, try the official step-by-step tutorials on the Godot docs.
