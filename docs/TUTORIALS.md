# Godot Tutorials & Exercises — Plate Harmony

Recommended learning path (short, focused):

1. Godot official beginner tutorials

- What: Scenes/nodes, signals, input, animation, export
- Link: https://docs.godotengine.org/en/stable/tutorials/index.html
- Exercise: Create a scene with a `CharacterBody2D` that moves using `ui_*` actions and prints position.

2. GDQuest — Beginner series (GDScript patterns)

- What: Project structure, signals, more robust controllers
- Link: https://www.gdquest.com/tutorial/godot/
- Exercise: Add sprite animations to the moving character and switch animations based on direction.

3. HeartBeast / YouTube project tutorial

- What: Hands-on small game loop, UI, and polish
- Link: search for "HeartBeast Godot tutorial" (YouTube)
- Exercise: Implement interactable objects (pick up an ingredient) and display a small HUD count.

Quick exercises for the two of you:

- Exercise A (programmer): Build the moving player scene in `res://scenes/player.tscn` and wire `ui_*` inputs.
- Exercise B (artist): Produce a 64x64 player sprite (PNG) with three frames for idle/walk and drop it into `assets/`.
- Exercise C (joint): Replace placeholder sprite with final art, add simple sound for steps, and iterate.
